# Proyecto-Final-de-Fullstack🚲 Bike Marketplace
A simple full‑stack application for posting, browsing, and managing bicycles for sale.
Users can create accounts, publish bikes with images, edit or delete their listings, and mark them as sold.
Admins have access to user and bike management tools.

⭐ Features
User registration and login (JWT)

Create, edit, delete and view bikes

Upload bike images (Cloudinary)

Mark bikes as sold

Admin dashboard (manage users and bikes)

Protected routes (frontend + backend)

🛠 Tech Stack
Frontend: React, Vite, React Router
Backend: Node.js, Express, MongoDB, Mongoose
Images: Cloudinary
